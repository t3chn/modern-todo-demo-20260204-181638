# Modern Todo (Demo)

Современный однопользовательский Todo-лист без зависимостей: чистые `index.html` + `style.css` + `app.js` и сохранение в `localStorage`.

## Запуск

```bash
python3 -m http.server 8000
```

Откройте: `http://127.0.0.1:8000`

## Возможности

- Добавление задач (кнопка или Enter)
- Выполнение/возврат (чекбокс)
- Удаление задач
- Фильтры: Все / Активные / Готово
- Очистка выполненных: кнопка или `⌘/Ctrl+Backspace`
- Сохранение между перезагрузками (localStorage)
